# songshow-kaida.com (甲斐田晴 楽曲検索ファンサイト)

にじさんじ所属VTuber「甲斐田晴」さんが関連する楽曲（オリジナル・カバー）を、印象や条件から検索できる非公式ファンサイトです。

[サイトはこちら](https://songshow-kaida.com)

## 🌟 プロジェクトの概要
「たくさんの楽曲の中から、お気に入りの一曲・今聞きたい一曲を探し出す手助けをしたい」という想いから制作しました。
賃貸物件検索のような感覚で、その時の気分やBPM、楽曲の持つ雰囲気から直感的に選曲できるシステムを目指しています。

## ✨ 特徴・仕様
- **詳細な印象検索**: 独自の印象値に基づいたフィルタリングが可能です。
- **重み付け表示**: 公式オリジナル楽曲が優先的に表示されるよう調整されています。
- **手動更新のこだわり**: BPMの計測やYouTube再生数の更新などを手動で行い、データの精度（体感値）を大切にしています。
- **ユーザー参加型**: Google Formsを通じて「曲の印象」を募集し、ファンの皆さんの声を反映させる仕組みを取り入れています。

## 🛠 技術スタック
- **Frontend**: HTML / CSS / JavaScript
- **Hosting**: GitHub Pages
- **Data Collection**: Google Forms
